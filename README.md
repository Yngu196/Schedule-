# Schedule 课表

基于 WakeUp 课程表 3.612 魔改重构的 Android 课表应用。

## 功能特点

- 📅 **多视图显示** - 支持周视图和今日视图切换
- 📥 **课程导入** - 支持从教务系统导入课程
- ⏰ **课前提醒** - 课程开始前自动推送通知
- 📱 **桌面小组件** - 今日课程/极简倒计时小组件
- ⏳ **倒计时** - 显示距离下节课的剩余时间
- 📖 **新手引导** - 首次使用提供引导教程

## 技术栈

- Kotlin + MVVM 架构
- AndroidX + Material Design
- Room + WorkManager
- 支持 Android 8.0+

## 开源说明

本项目是对 WakeUp 课程表 3.612 版本的魔改重构，开源旨在降低后来者的门槛。教务解析使用了 CourseAdapter 库。
